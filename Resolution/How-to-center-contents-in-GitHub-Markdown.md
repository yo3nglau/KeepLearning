## Resolution: How to center contents in GitHub Markdown

### Paragraph tag of HTML

#### Step 1

Type `<P>` tag to create a paragraph tag.

#### Step 2

Use the align attribute to set the align method.

```python
<p align="center">
	Enjoy learning new knowledge.
</p>
```

### Example

<p align="center">
	Enjoy learning new knowledge.
</p>
